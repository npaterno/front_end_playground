R and RStudio

"R is ‘GNU S’, a freely available language and environment for statistical computing and graphics which provides a wide variety of statistical and graphical techniques: linear and nonlinear modelling, statistical tests, time series analysis, classification, clustering, etc" - CRAN

Software

For the computer lab part of our course, we will be learning the basics of R. In order to follow along in each lesson and complete assignments, you'll need to download and install two pieced of software: R and RStudio. Follow the links below to get started or skip to your operating system below. Note: links will open in a new tab.

  1. R
  2. RStudio

Windows 10

Once you've navigated to the Comprehensive R Archive Network page (link #1 above), select the "Download R for Windows" link.

Next, select the "install R for the first time".

Then choose "Download R 4.0.4 for Windows". Note: This is the current version of R as of 2/16/21. If a newer version becomes available the number used will be different from 4.0.4.

Now we're ready to download RStudio. This is the interface we'll use to write and execute our code. From the link above, select "Download RStudio for Windows".

Mac

Once you've navigated to the Comprehensive R Archive Network page (link #1 above), select the "Download R for (Mac) OS X" link.

To complete the next step, you'll need to know what version of OS X you have. Click on the Apple icon on the top left of your screen.

Select "About this Mac"

Your OS X version will be listed on this page.

Back on the CRAN site from the end of step 1, select the version of R that matches your OS X. Note the image shows what I would select since my Mac has an older version of OS X.

Now we're ready to download RStudio. This is the interface we'll use to write and execute our code. From the link above, select "Download RStudio for Mac".
